# Planner Agent

Агент для планирования и декомпозиции задач.

## Описание

Этот агент помогает разбивать большие задачи на более мелкие, планировать архитектуру и определять последовательность реализации.

## Использование

```
@planner plan authentication module
@planner decompose task parsing feature
@planner design database schema for new feature
```

## Возможности

- Анализ требований из Specification.md
- Декомпозиция задач на подзадачи
- Планирование последовательности разработки
- Определение зависимостей между задачами
- Оценка сложности и рисков

## Контекст

Агент имеет доступ к:
- Всей документации проекта (docs/)
- Architecture.md для понимания текущей архитектуры
- Pseudocode.md для понимания алгоритмов
- Specification.md для требований

## Выходной формат

Агент создаёт план в виде:
1. Список задач с приоритетами
2. Зависимости между задачами
3. Рекомендуемая последовательность
4. Потенциальные риски и edge cases
